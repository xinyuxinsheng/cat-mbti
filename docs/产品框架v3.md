# 喵格测试 · 产品框架 v3（待确认稿）

> 2026-07-03 · 基于调研与多轮讨论收敛，确认后按「版本路线」开工

---

## 1. 产品定位与商业模式

**一句话**：猫咪人格测试 + 晒猫素材工厂。

**目标用户**：中国养猫宠物主（约 50% 田园猫、20% 英短的用户结构）。

**商业闭环**：免费测试制造分享 → 分享海报/配对卡带回新流量 → 单一付费点 ¥9.9（深度报告 + AI 专属卡片打包）→ 二次生成微交易增收。

**行业基准**（调研结论）：测评类付费转化率 ≈ 2%；主流定价 9.9~29.9；付费墙标配 = 划线原价 + 解锁清单 + 数据背书。

---

## 2. 三条设计原则

1. **答题全程零付费暗示**——付费信息只在结果页出现（沉没成本效应，调研验证的转化关键）。
2. **皮芯分离**——产品皮肤是轻松愉快的暖色贴纸风（测试像玩小游戏）；付费商品（AI 卡片）是写实电影风。可爱的盒子开出高级的货。
3. **单付费点**——MVP 只有一个 ¥9.9 SKU，所有付费欲望汇到一处，转化数据干净。

## 3. 模块详解

### 模块一：首页与测试（免费）

| 内容 | 说明 |
|---|---|
| 主视觉 + CTA | 猫咪 IP + 开始测试 |
| 价值说明模块 | 三条"测完你会得到"：读懂迷惑行为 / 相处指南 / 专属人格海报 |
| 16 类型图鉴 | 四大派系分区展示，制造"我家猫是哪个"好奇缺口；小程序版做收集图鉴（测过点亮） |
| 隐私协议 | 首页"开始即同意"轻提示；小程序版配《隐私保护指引》+ 官方授权弹窗 |
| 建猫入口 | 猫名 + 品种/花色（14 项混合列表）。**不传照片**——照片在付费后为生成卡片时上传 |
| 答题 | 24 题五档（4 道支配性题已改单猫/多猫通用）；中途退出自动保存 + 挽留弹窗（"豆豆的性格画像已完成 60%…"） |
| 报告生成动画 | 3~5 秒"正在分析 24 项行为数据"，制造报告价值感 |

### 模块二：报告 + AI 卡片（唯一付费点 ¥9.9）

**免费层**（够发朋友圈，全程带猫名）：
- 类型名 + 猫名（"科学家猫 · 豆豆"）、简笔画形象、4 标签、性格简介、双极维度条、标志语录
- 免费海报（简笔画版，带小程序码/链接）

**付费层 ¥9.9 打包**：
1. **深度报告**：行为翻译词典 / 相处红黑榜 / 喂养建议 / 完整维度解读——16 套预写静态内容，付费时刻零额外输入
2. **AI 专属卡片**：付费后引导上传照片（此时弹隐私授权：用途+删除时限；引导"传一张看镜头、光线好的照片"）→ 写实电影风画像（猫咪头 + 类型服装道具 + 品种花色 + 性格关键词进提示词）→ 卡片 = 画像 + "科学家 kkk" + 关键词 + 语录。**每次生成出 2 张供选，含 1 次免费重生成**
3. **二次生成微交易**：¥3.9~4.9/次（不满意再来 / 想要更多张）

**付费墙配置**：划线价 ¥39、解锁清单逐项打勾、"基于 X 万只猫咪行为数据"背书、限时标签。

**成本红线**：生图单均成本 1~3 元；上线后盯"平均每单生成次数"，>2.5 次即调价或调张数。

### 模块三：配对（纯免费裂变引擎）

- 入口：结果页"测测和好友猫咪的缘分" + 档案页常驻
- 流程：分享卡带类型参数 → 好友测完自动出 相性分 + 一句锐评 + 双猫合影海报
- 数据：16 类型两两组合 136 条相性数据，静态写死，无后端成本
- ⚠️ 合规红线:不做"分享解锁"式诱导分享(微信处罚项);分享动机靠内容"值得晒"

### 模块四：猫咪档案（免费留存）

猫信息 / 测试历史 / 已购收藏 / 配对记录 / 图鉴点亮。多猫建档免费测试免费（多猫用户是付费主力）；重测免费或前置激励视频广告。

### 模块五：支付与订单

微信支付（需个体工商户 + 商户号，尽早启动办理）；订单记录；兑换码体系预留（后期分销）。iOS 虚拟支付规避：iOS 端隐藏价格入口，引导 H5 支付。

### 模块六：增长与数据

- 全漏斗埋点：进入 → 开始 → 完测 → 看报告 → 点付费 → 支付成功 → 上传照片 → 卡片生成 → 分享 → 回流
- 带参二维码/分享卡追踪回流来源
- 订阅消息仅一个场景：卡片生成完成通知
- 公众号联动沉淀私域；品种×类型统计彩蛋反哺内容投放（"橘猫里 38% 是霸总猫"）

**明确不做**：社区 feed、订阅制、AI 追问问答、分享解锁门槛。

## 5. 核心风险与验收线

1. **AI 卡片相似度 = 付费转化生死线**。用户买的是"我家猫成为电影主角"，不像=货不对板。
   - 选型唯一标准：图生图宠物相似度保持能力
   - 验收线：20 个真实猫主盲测"像不像你家猫"，通过率 <80% 不上线该付费点（先只卖纯报告）
2. **9.9 打包毛利偏薄**：对策见模块二成本红线。
3. **微信合规**：诱导分享红线、虚拟支付 iOS 限制、隐私弹窗审核要求。

## 6. 待升级资产：16 个猫咪形象

现有 16 个 SVG 简笔画偏粗糙，**H5 验证期够用，小程序上线前替换**（它们是首页图鉴的门面）。

- 技术上已解耦：代码通过 `catSVG(code)` 引用，替换为图片文件即插即用
- 候选方案：① AI 批量生成统一风格角色集（成本低、速度快，需人工挑选保持一致性）② 约插画师定制（贵但独占）③ 两步走：先 AI 版上线，反响好再定制
- 风格约束：保持暖色贴纸风（产品皮肤的一部分），与写实卡片形成档次差

## 7. 版本路线

| 阶段 | 载体 | 范围 | 验证什么 |
|---|---|---|---|
| **V0.5 漏斗验证** | H5（现有站升级） | 首页改版（图鉴/价值模块/隐私）+ 报告分层 + 假付费按钮（点击即记录+留资）+ 埋点 + 免费配对 | 完测率 ≥80%？付费点击率 ≥5%？配对分享率？ |
| **V1.0 收入验证** | H5 + 云函数 | 接真支付（虎皮椒/个人通道）+ AI 卡片全链路 + 深度报告上线 | 真实付费转化 ≥2%？卡片相似度过验收线？ |
| **V2.0 小程序 MVP** | 微信小程序 | 全量迁移 + 微信支付 + 档案 + 图鉴收集 + 订阅消息；替换 16 形象 | 小程序生态内的自然流量与回流效率 |
| **P2 候选池** | — | 相处剧本付费版 / 主人×猫配对 / 多猫相性 / 兑换码分销 | 数据触发才启动 |

**并行事项**（不阻塞开发）：个体工商户注册（周期最长，立即启动）；生图 API 选型测试；16 形象升级。

## 8. 关键指标看板

- 北极星：**每周付费单数**
- 漏斗：完测率 ≥80% / 付费转化 ≥2% / 分享率 / 回流系数（每 100 个分享带回多少新用户）
- 商品健康：卡片相似度通过率 ≥80% / 平均每单生成次数 ≤2.5 / 二次生成购买率
