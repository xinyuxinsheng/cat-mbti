## 一、计分规则（对应 cats.js 中 calcScores / getTypeCode）

| 规则项 | 当前设定 |
|---|---|
| 答案选项 | 非常不符合=1 / 不太符合=2 / 不确定=3 / 比较符合=4 / 非常符合=5 |
| 正向题 | 得分 = 原始分（越同意 → 越偏向该维度高分侧 E/N/T/J） |
| 反向题 | 得分 = 6 − 原始分（越同意 → 越偏向低分侧 I/S/F/P） |
| 归一化 | 每维度得分 =（总分 − 题数）/（题数 × 4），范围 0~1，全选"不确定"= 0.5 中线 |
| 判型 | 得分 > 0.5 → E/N/T/J；≤ 0.5（含平局）→ I/S/F/P（平局判给温和侧） |
| 维度含义 | EI=社交性（E外向/I内向）· SN=好奇心（N好奇/S谨慎）· TF=支配性（T强势/F温和）· JP=规律性（J规律/P随性） |

## 五、已知待审核问题（我先替你标出来的）

1. **冤家对手大量重复 ESTJ/ESTP**：16 个类型里有 12 个的冤家包含"霸总猫 ESTJ"，SJ 派 4 个类型的冤家里都有"侦探猫 ESTP"——分布明显不均，建议人工重新分配，让每个类型被当冤家的次数大致均衡（每类型 1~2 次）。
2. **最佳拍档同样有集中现象**：ISTP/ISFP、ESTP/ESFP 等组合出现频次偏高，可一并调整。
3. **ISFJ 叫"学者猫"但分在实干派**：学者的意象和"实干"略有错位，可考虑改名（如"观察家猫"）或维持现状。
4. **Q11（家里跑酷）计入好奇心 N**：跑酷更像精力/冲动，与 Q24（注意力短→P）概念有重叠，审题时可留意。
5. 优缺点每类型各只有 2 条，海报和报告展示时略显单薄，如要加到 3 条，直接在表格里加即可。
