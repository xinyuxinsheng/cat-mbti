# 喵格测试 🐱 猫咪 MBTI 性格测试

24 道题，发现你家猫主子的隐藏性格。16 种猫咪人格类型等你解锁！

**在线体验：https://xinyuxinsheng.github.io/cat-mbti/**

## 功能

- 🐾 24 道原创中文题目，覆盖社交性 / 好奇心 / 支配性 / 规律性四个维度
- 😺 16 种猫咪人格类型（霸总猫、科学家猫、面包师猫……），每种配专属简笔画
- 📊 MBTI 双极维度条，直观展示每个维度的偏向
- 📸 一键生成分享海报，长按保存发朋友圈
- 📱 移动端优先设计，微信内置浏览器可用

## 技术

纯 HTML + CSS + JavaScript，零依赖，无后端。SVG 手绘猫咪插图，Canvas 生成海报。

## 本地运行

直接用浏览器打开 `index.html` 即可。
